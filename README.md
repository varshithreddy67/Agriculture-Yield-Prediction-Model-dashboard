## 📌 Project Overview

This project focuses on analyzing and predicting agricultural crop yield using historical farming and environmental data. A **Power BI interactive dashboard** was developed to visualize key factors influencing crop yield and to support data-driven decision-making for farmers, researchers, and policymakers.

The dashboard provides insights into crop performance across regions, seasons, and environmental conditions, helping identify patterns and trends that affect agricultural productivity.


## 🎯 Objectives

* Analyze historical agriculture data to understand yield patterns
* Identify key factors influencing crop yield (rainfall, area, season, region, etc.)
* Predict crop yield trends using data-driven insights
* Visualize regional and crop-wise yield performance
* Enable better planning and decision-making in agriculture


## 📊 Dashboard Features

* **Crop-wise Yield Analysis** – Compare yield across different crops
* **Region/State-wise Insights** – Identify high and low productivity areas
* **Seasonal Trends** – Analyze yield variations across seasons
* **Year-wise Yield Trends** – Understand long-term agricultural performance
* **Interactive Filters & Slicers** – Dynamic exploration of data


## 🛠️ Tools & Technologies Used

* **Power BI** – Data modeling, visualization, and dashboard creation
* **Excel Dataset** – Agriculture yield prediction dataset
* **DAX** – Calculated measures and KPIs
* **Power Query** – Data cleaning and transformation

## 📁 Dataset Description

The dataset contains historical agricultural data with attributes such as:

* Crop Name
* Year
* Season
* State/Region
* Area under cultivation
* Production
* Yield


## 🔍 Key Insights

* Certain crops show consistent yield growth over time
* Seasonal and regional factors significantly impact productivity
* Yield variation highlights opportunities for improved farming practices
* Data visualization helps quickly identify high-risk and high-performing area


## 📈 Future Enhancements
